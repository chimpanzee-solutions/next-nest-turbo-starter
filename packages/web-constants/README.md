# `@repo/web-constants`

Shared frontend constant data for the web apps.

## Overview

This package exposes reusable constant sets used across the web surfaces.

Current exports include:

- cities
- currencies
- date formats
- timezones

Import from the package root:

```ts
import { timezones, currencies } from '@repo/web-constants';
```

## Scope

Use this package for stable shared reference data that needs to be reused by more than one app.
