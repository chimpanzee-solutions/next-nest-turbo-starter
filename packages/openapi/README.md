# `@repo/openapi`

Generated HTTP client and TanStack Query hooks for the starter API.

## Inputs and outputs

- Spec: [apps/api/swagger-spec.json](../../apps/api/swagger-spec.json)
- Config: [orval.config.ts](./orval.config.ts)
- Output: [src/generated.ts](./src/generated.ts)
- Client wrapper: [src/api-client.ts](./src/api-client.ts)

## Regenerate

```sh
pnpm run codegen
```

Or:

```sh
pnpm --filter @repo/openapi run codegen
```
